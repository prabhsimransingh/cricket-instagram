📖 Setup Guide

1. Instagram

Convert to Creator/Business → connect Facebook Page → Meta Developer App → Instagram Graph API → long-lived token with instagram_content_publish

2. Groq (AI Captions)

Free at console.groq.com → API Keys. Powers all caption generation using Llama 3.3.

3. Imgur (Image Hosting)

Register free at api.imgur.com — use "OAuth 2 without callback". Copy the Client ID.
